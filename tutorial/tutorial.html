Downloading MetaMask

Using Google Chrome ? Follow this link to download the extension.
Add the extension to Chrome by clicking on the “Add to Chrome” button.
Confirm the download by clicking on the “Add extension” button from the pop-up.

Using Microsoft Edge ? Follow this link to download the extension.
Add the extension to Chrome by clicking on the “Add” button.
Confirm the download by clicking on the “Add extension” button from the pop-up.

Using Mozilla Firefox ? Follow this link to download the extension.

Import test Accounts

First time with MetaMask ?

If this is the first time you are using MetaMask, click on the "Start" button and "Import the Wallet". Click on "import using Secret Recovery Phrase" from the pop-up.

fame elevator saddle renew match guilt seat duck seven buyer true smoke

Then choose a password, check "I have read and agree to the and press Terms of Use" and click on “Import”.

Already used MetaMask before ?

If you have already used MetaMask before, the importation steps are a little bit different.
If you are currently logged in, you can either import the test accounts as described above, or log out.

To log out, click on the fox, then on the circle on the top right of the pop-up, and click on "Disconnect".
Click again on the fox, then click on the dropdown menu and select "Ropsten Test Network".
Now click on "import using Secret Recovery Phrase".
In the Wallet secret Recovery Phrase form, paste the following:

fame elevator saddle renew match guilt seat duck seven buyer true smoke

Then choose a password and press on “Restore”. The usable test accounts have now been imported.

Choose Ropsten Test Network

In order to use the NibbleUI, you must be running on th "Ropsten Test Network".
To do so, click on the fox, then click on the dropdown menu and select "Ropsten Test Network".

Switch Between Test Accounts

In order to use Nibble, your account needs to have ETH.
We have set up several usable accounts on the "Ropsten Test Network".
For those account, we provided a certain amount of ETH.
But as they can be used be everybody, it may be possible that one or more accounts run out of ETH.
In this case, you may want to use another imported account.

To switch from one account to another, click on the fox and then on the circle at the top right of the pop-up.
You can now select an account from the list.
For each and single one of them, you should be able to see their current ETH balance.
If you only have one account in the list, go back to the "Import test Accounts" step.

Get the Current Wallet Address

In order to get the current account address, click on the fox, then click on account name.
The account address will then be copied to clipboard. Your address is now ready to be pasted as a recipient.

Connect the NibbleUi with MetaMask

Finally, in order to be able to use the NibbleUi, you will need to connect the site with MetaMask.
To do so, go on the main page an select the bank or user UI.
Click on the "Enable access" button.
On the "Connect with MetaMask" pop-up, select all accounts and click on "next".
Last, click on "Connect"

You should now be able to use and play with the NibbleUI !
Have fun !!
